# StandX Trading Dashboard

A fully client-side trading-stats dashboard for [StandX](https://standx.com) perps. Upload your trade export and get realized PnL, win rate, profit factor, per-market breakdown, and a shareable card — all parsed locally in your browser, no backend, no login.

**Community project — not officially affiliated with StandX.**

## Quick start

The app lives in [`app/`](app):

```bash
cd app
npm install
npm run dev
```

See [`app/README.md`](app/README.md) for full docs and [`app/SECURITY.md`](app/SECURITY.md) for the privacy/security model.

`project/` contains the original Claude Design handoff this was built from.
